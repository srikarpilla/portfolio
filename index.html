Project 2

Key Point of Project

The Speech Emotion Recognition (SER) system was developed to accurately identify and classify emotions from audio samples, enhancing applications in customer service, mental health assessments, and human-computer interaction. The system used machine learning algorithms like SVM, Random Forests, and LSTM networks, trained on diverse datasets like RAVDESS and TESS. Feature extraction techniques were employed to extract relevant features like Mel-frequency cepstral coefficients and prosodic features. Performance evaluation metrics were used to assess the model's accuracy, precision, recall, and F1 score. The SER system can be integrated into various applications to facilitate more natural emotional interactions.
